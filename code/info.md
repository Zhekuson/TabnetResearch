# Code

## Навигация
1) [Докер-файл](Dockerfile)  
Служил для сборки и запусков на сервере ФКН  
`docker build . -f Dockerfile -t <>`
Запуск образа с маунтом (-v) корневой директории (этой)  
Все пути прописываются в [конфиге](config.json)
2) [pipelines](pipelines)  
Пайплайны для всех моделей
3) [augmentations](augmentations)  
Файлы для аугментаций и перебора гиперпараметров для них
4) [utils](utils)  
Пакет для загрузки датасетов, генерации данных, и прочего
5) [experiments](experiments)
Файлы с запуском экспериментов
